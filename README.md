# Airline-Flight-Delay-Performance-Analytics-SQL-Power-BI

Airline Flight Delay & Performance Analytics

Project Overview : 
This project analyzes airline operational performance using over 100,000 flight records. The analysis focuses on flight delays, cancellations, airport efficiency, route performance, and delay root causes. SQL was used for data exploration and KPI analysis, while Power BI was used to build an interactive dashboard for operational insights.

Business Problem :
Airlines face significant operational challenges due to flight delays and cancellations. The objective of this project is to identify performance bottlenecks, understand delay patterns, evaluate airport and airline efficiency, and uncover the major causes of delays to support data-driven decision-making.
